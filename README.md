# Learning Azure SQL & Cosmos DB (NoSQL)

## Learning Points
* Azure SQL (cloud databases)
* Azure Functions (Javascript) with Azure SQL
* JSON column data type
* Blob (varbinary) column data type
* Additional logins and users in Azure SQL
